# CodeReviewAI
CodeReviewAI
###### Works fine with:

    - internet connection required
    - git:latest
    - docker: Docker latest
    - docker-compose latest

### INSTALATION

1. Clone repository:

```
git clone <repo url>
cd CodeReviewAI
```

2. Add .env file
```
OPENAI_API_KEY=your_key
GITHUB_TOKEN=your_key
```
4. Run repository

cd <path/to/local/repo>
```
docker-compose up --build
```

4. Open
```
   http://0.0.0.0:8000/
```
